# Smart Fisherman System

## Overview

Smart Fisherman System is a modern platform designed to help fishermen make better decisions by providing important information such as weather updates, fishing locations, fish market prices, and emergency support services.

The goal of this project is to improve safety, increase productivity, and help fishermen maximize their income through technology.

---

## Backend Development

The backend of this system was developed using **Spring Boot** and follows RESTful API principles to provide secure and efficient communication between the frontend and the database.

### Technologies Used

* Java 21
* Spring Boot
* Spring Data JPA
* Spring Security
* PostgreSQL
* Maven
* REST API
* Git & GitHub

---

## Features

* User Registration and Login
* Weather Information API
* Fish Market Price Management
* Fishing Location Recommendations
* Emergency SOS Support
* Secure Authentication and Authorization
